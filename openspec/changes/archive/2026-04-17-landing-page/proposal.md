## Why

Fabrick needs a public-facing presence. Before any product ships, there should be a place that communicates what Fabrick is, builds early interest, and collects waitlist signups. A landing page is the minimal credible starting point.

## What Changes

- Add `applications/landing/` — a standalone Vite + React + Tailwind app
- Landing page with sections: Hero, Problem, Solution, Privacy, Footer
- Dark theme, AI-first aesthetic
- Runs fully locally via `npm run dev`

## Capabilities

### New Capabilities

- `landing-page`: Public marketing page that communicates Fabrick's value proposition, target audience, and privacy model

## Impact

- New directory `applications/landing/` added to repo
